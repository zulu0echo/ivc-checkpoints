# results/

Measurement outputs (regenerated by the harness):

- `prover.json` — prover-side [M] (fold time/step, decider prove time, peak RSS, calldata
  bytes, revm gas). Written by `cargo run -p bench`.
- `forge_gas.json` — on-chain [M] (NovaDecider deploy gas, `verifyNovaProof` tx gas,
  `settleEpochProven` tx gas, calldata bytes). Written by the Foundry `GasTest`.
- `measured.json` — merged, provenance-labeled. Written by `script/compare_to_model.py`.
- `comparison.md` — measured [M] vs the analytical model [A], flagging any `verifyNovaProof`
  deviation > 5%. Written by `script/compare_to_model.py`.

The checked-in copies were produced on the small (`light-test`) path on a 24 GB machine —
see README §Results and §Hardware. Regenerate with `--features full-bench` on ≥64 GB for
production figures.
